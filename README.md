# What to do?
A simple To-Do List project submitted for The Odin Project course.

> [!TIP]
> Need help setting up TailwindCSS with Webpack?  
> Check out this [detailed guide](https://gist.github.com/mkgp-dev/66d2f158057c539dad55c24804f66f82) for step-by-step instructions or use my [boilerplate](https://github.com/mkgp-dev/webpack-tailwindcss-v4-boilerplate).

## Features
- Uses [localForage](https://localforage.github.io/localForage/) for efficient Web Storage API handling
- Built with [TailwindCSS](https://tailwindcss.com) for styling and [Heroicons](https://heroicons.com) for icons
- Lightweight and simple Task Manager interface
- Optimized for desktop viewing

## Goals
- Improve JavaScript functionality
  - Refactor class-based code into simpler function-based structures
- Improve Front-End design
  - Add smooth transition effects to make the UI look more modern
  - Enhance mobile compatibility and responsive layout
- Add more features
  - Implement data export/import functionality 
  - Integrate a proper database with user authentication for better data handling
  - Implement smart notifications to remind users of upcoming deadlines

## Deployment
Install required libraries
```bash
  npm install
```

Build the project using **Webpack** or

```bash
  npm run build
```

Safely run the project in development

```bash
  npm run dev
```
